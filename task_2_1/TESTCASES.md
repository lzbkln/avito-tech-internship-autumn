# Тест-кейсы для API объявлений

## Общая информация

Данный файл содержит тест-кейсы для проверки четырех ручек:

1. Создать объявления
2. Получить объявления по его идентификатору
3. Получить все объявления по идентификатору продавца
4. Получить статистику по айтем id

---

## 1. Создание объявления (`POST /api/1/item`)

Для `sellerID` выделим 3 класса эквивалентности:

1. Значения ниже минимального допустимого (`sellerID < 111111`).
2. Значения в допустимом диапазоне (`111111 <= sellerID <= 999999`).
3. Значения выше максимального допустимого (`sellerID > 999999`).

Граничные значения для `sellerID`:

- Нижняя граница: `111110` (ниже минимального допустимого).
- Минимальное допустимое значение: `111111`.
- Максимальное допустимое значение: `999999`.
- Верхняя граница: `1000000` (выше максимального допустимого).
